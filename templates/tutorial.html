Panduan Pasang Modifikasi

Langkah 01
Unduh Berkas Valid

Pilih paket modifikasi resmi kustom dari beranda RexCraft Hub, masuk ke Tab Daftar Versi, lalu klik tombol download untuk mendapatkan berkas berekstensi .zip, .jar, atau .mcpack.

Langkah 02
Pindahkan Berkas (Minecraft Java)

Buka direktori penyimpanan run game Anda di %appdata%/.minecraft/mods lalu paste file .jar yang sudah diunduh ke dalam folder tersebut.

Langkah 03
Eksekusi Impor (Minecraft MCPE / Bedrock)

Untuk pengguna Android/iOS/Windows 10, cukup klik berkas berkode .mcpack atau .mcaddon yang selesai diunduh agar game Minecraft otomatis melakukan import secara langsung.
