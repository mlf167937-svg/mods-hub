Panduan Pasang Modifikasi

Langkah 01
Unduh Berkas Valid

Pilih paket kustom resmi dari beranda RexCraft Hub, masuk ke Tab Daftar Versi, lalu tekan tombol unduh untuk mendapatkan berkas berekstensi .zip, .jar, atau .mcpack.

Langkah 02
Pindahkan Folder (Minecraft Java)

Buka folder run game Anda di direktori %appdata%/.minecraft/mods lalu simpan file .jar yang sudah diekstrak ke dalam folder tersebut.

Langkah 03
Eksekusi Impor otomatis (Minecraft MCPE / Bedrock)

Untuk pengguna mobile/Android/iOS, cukup klik berkas berkode .mcpack atau .mcaddon yang selesai diunduh agar sistem game Minecraft melakukan impor aset secara langsung.
