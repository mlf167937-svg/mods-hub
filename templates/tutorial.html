Tutorial Pemasangan Berkala

Langkah Instalasi Java Edition

  1. Pastikan Anda telah menginstal client loader (Fabric / Forge) yang sesuai.
  2. Download mod racikan dari repositori RexCraft Hub.
  3. Tekan tombol WIN + R pada keyboard Anda, ketik %appdata%, dan tekan Enter.
  4. Buka folder .minecraft lalu cari folder bernama mods. (Jika tidak ada, buat folder baru dengan nama 'mods').
  5. Paste atau pindahkan file .jar yang telah didownload ke dalam folder tersebut.
  6. Buka Minecraft Launcher, pilih profil loader yang tepat, dan klik Play.

Langkah Instalasi MCPE (Bedrock)

  1. Download file dari repositori kami.
  2. Buka aplikasi File Manager atau ZArchiver di HP Anda.
  3. Cari file yang baru saja didownload.
  4. Jika file berformat .mcpack atau .mcaddon, cukup klik file tersebut dan pilih opsi "Buka dengan Minecraft".
  5. Jika file berformat .zip, Anda harus mengekstraknya terlebih dahulu. Pindahkan folder hasil ekstrak ke dalam direktori games/com.mojang/behavior_packs (jika itu mod/behavior) atau ke resource_packs (jika itu shader/tekstur).
  6. Buka Minecraft, masuk ke pengaturan dunia, dan aktifkan pack tersebut di bagian Add-Ons.
