📖 Panduan Pemasangan Berkas

Ikuti langkah-langkah di bawah ini untuk memasang Mod atau Shader ke dalam Minecraft kamu.

☕

Minecraft Java Edition (PC)

Langkah 1: Unduh berkas mod berformat .jar dari RexCraft Mods Hub.
Langkah 2: Tekan tombol Windows + R di keyboard kamu, ketik %appdata%\.minecraft, lalu klik OK.
Langkah 3: Cari folder bernama mods. Jika belum ada, silakan buat folder baru dengan nama mods (huruf kecil semua).
Langkah 4: Pindahkan berkas .jar yang sudah kamu unduh tadi ke dalam folder mods tersebut.
Langkah 5: Jalankan Minecraft Launcher kamu dan pastikan kamu memilih profile **Forge** atau **Fabric** sesuai dengan kebutuhan mod tersebut. Selamat bermain!
📱

Minecraft Bedrock / MCPE (Android/iOS/Win10)

Langkah 1: Unduh berkas berformat .mcpack, .mcaddon, atau .zip dari web ini.
Langkah 2 (Jika format .mcpack / .mcaddon): Kamu cukup buka aplikasi File Manager di HP kamu, lalu **klik langsung berkas tersebut** dan pilih untuk dibuka menggunakan game **Minecraft**. Game akan otomatis mengimpor berkas secara instan!
Langkah 3 (Jika format .zip): Ekstrak folder tersebut terlebih dahulu menggunakan aplikasi seperti ZArchiver.
• Pindahkan folder **Resource Pack** ke: Android/data/com.mojang.minecraftpe/files/games/com.mojang/resource_packs
• Pindahkan folder **Behavior Pack** ke: Android/data/com.mojang.minecraftpe/files/games/com.mojang/behavior_packs
Langkah 4: Masuk ke dalam game Minecraft, buat dunia baru atau edit dunia yang ada, lalu aktifkan Resource/Behavior Pack tersebut di menu pengaturan dunia sebelum menekan tombol Play.
